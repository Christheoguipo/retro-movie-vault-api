Retro Movie Vault API Documentation

This API provides a list of information about classic movies, directors and genres.

Description URL Request Query Parameters Request Body Data Format Response Format
Returns a list of ALL movies. /movies GET None. None. A JSON object holding data about all the movies. Example: 
[{
  "title": "Movie Title",
  "description": "Movie Description.",
  "genre": "Crime",
  "director": "Quentin Tarantino",
  "image_url": "https://website.com/Image.jpg"
}]
Returns data about a single movie by title. /movies/:title GET title - The title of the movie. None. A JSON object holding data about a single movie, containing a title, description, genre, director and image url. Example: 
[{
  "title": "Movie Title",
  "description": "Movie Description.",
  "genre": "Crime",
  "director": "Quentin Tarantino",
  "image_url": "https://website.com/Image.jpg"
}]
Returns data about a genre by movie title. /movies/:title/genre GET title - The title of the movie. None. A JSON object holding data about a single movie, containing the genre. Example: [{"genre": "Crime"}]
Returns data about a director by name. /directors/:name GET name - The name of the director. None. A JSON object holding data about a director, containing bio, birth year and death year. Example: 
[{
  "bio": "Details about the director.", 
  "birth_year": "1950", 
  "death_year": "N/A"             
}]
Allow new users to register. /register POST None. A JSON object holding data about a new user to register, structured like: 
[{
  "username": "yourusername", 
  "password": "yourpassword", 
  "email": "youremail@gmail.com"
}]
A text message indicating whether the user was successfully registered.
Allow users to update their user info. /:username PUT username - The username of the user to be updated. A JSON object holding data about user's info to update, structured like: [{"username": "your_new_username"}] A text message indicating whether the user info was successfully updated.
Allow users to add a movie to their list of favorites. /:username/favorites POST username - The username of the user. A JSON object holding data about user's favorite movie to be added to favorites, structured like: 
[{
  "title": "Movie Title",
  "description": "Movie Description.",
  "genre": "Movie Genre",
  "director": "Movie Director",
  "image_url": "https://website.com/Image.jpg"
}]
A text message indicating whether the movie was successfully added to the user's favorites.
Allow users to remove a movie from their list of favorites. /:username/favorites/:title DELETE username - The username of the user.
title - The title of the movie. 		None. A text message indicating whether the movie was successfully deleted from the user's favorites.
Allow existing users to deregister. /:username DELETE username - The username of the user to deregister. None. A text message indicating whether the user was successfully deregistered.
